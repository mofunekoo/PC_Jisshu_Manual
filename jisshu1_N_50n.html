実習1 課題2: 短チャネル MOSFET のシミュレーション

チャネル長が短い NMOS トランジスタの特性を確認します。最初に作成した NMOS の回路を利用し、チャネル長が 50nm、チャネル幅が500nm のトランジスタのシミュレーションを実施します。

1. トランジスタのパラメータ設定

NMOSトランジスタのシンボル上にカーソルを置き、右クリックしてパラメータ設定ウィンドウを表示します。以下のパラメータを入力します。

  • チャネル長 (L): 50nm
  • チャネル幅(W): 500nm

2. DC 解析の準備

メニューの Simulate > Edit Simulation Cmd を選択し、「DC Sweep」を設定します。

1st Source: VDS (V2)

2nd Source: VGS (V1)

開始値、終了値、増加分を設定して解析の準備を行います。

.dc V2 0 1 0.01 V1 0 1 0.2

3. 回路図ファイルの保存

作成した回路図をシミュレーションする前に、必ず正しい手順で保存を行います。

1. 保存の手順

File > Save As を使用し、「ドキュメント > LTspice 実習フォルダ」に保存します。必ず Save As を使用して保存を行い、ツールバーの「保存」ボタンで自動生成される Draft○.asc などの名前を使用しないようにしてください。

2. ファイル名の命名規則

ファイル名は指定された形式に従ってください。例えば、N_50n.ascやP_1u.ascなど、実習ごとに指定された名前を使用します。

3. 上書き保存をしない

保存を行う際、上書き保存をしないように注意してください。上書き保存をすると、過去のデータや回路が消えてしまう可能性があります。異なるバージョンを保存する場合は、新しい名前で保存するようにしてください(例: N_50n_ver2.asc)。

4. 保存後の確認

保存した回路図ファイルが指定フォルダに正しく保存されているか確認します。これを怠ると、次回の実習時にファイルが見つからず、作業が中断されることがあります。
